# System Context

## Project Identity

This workspace uses a **technology-agnostic agentic framework** to coordinate AI-assisted software development. The framework defines roles (agents), reusable capabilities (skills), orchestrated processes (workflows), and shared standards (rules).

## Framework Philosophy

- **Agents** are specialized personas with defined responsibilities and access to specific skills.
- **Skills** are modular, reusable capabilities that can be invoked by one or more agents.
- **Workflows** are orchestrated sequences where a lead agent coordinates others to achieve a goal.
- **Rules** are workspace-wide standards that all agents must follow at all times.
- **Production artifacts** are documents generated by workflows (specs, PRDs, reports) and stored in `production_artifacts/`.
